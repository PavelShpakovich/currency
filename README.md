# Курс доллара к BYN

Простой Next.js экран для браузера на TV. Экран не требует взаимодействия: он сам переключает карточки с курсами и обновляет данные каждую минуту.

## Что показывает

- официальный курс USD/BYN от НБРБ
- банки Бреста по USD один за другим
- логотип банка и курс покупки / продажи

## Проверенные API

- `https://api.nbrb.by/exrates/rates/USD?parammode=2`
- `https://myfin.by/currency/usd/brest` через текстовый mirror `https://r.jina.ai/http://...`

## Запуск

```bash
npm run dev
```

Затем откройте `http://localhost:3000`.

## Поведение экрана

- автообновление данных: каждые 60 секунд
- автопереключение карточек: каждые 10 секунд
- формат: TV-first, крупная типографика, без кнопок и ручного управления

## Основные файлы

- `app/page.tsx` — серверная точка входа
- `app/api/rates/route.ts` — агрегированный внутренний API
- `components/tv-rates-board.tsx` — клиентский экран с автообновлением и автоциклом
- `lib/sources/nbrb.ts` — адаптер НБРБ
- `lib/sources/myfin.ts` — парсер банков Myfin для Бреста
- `lib/sources/index.ts` — сборка единого снимка данных
